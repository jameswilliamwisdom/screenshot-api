Screenshot API

A simple API for taking screenshots of web pages.

How x402 Micropayments Work

This API uses the x402 protocol for micropayments. This means:

  • Pay-per-request: You only pay for what you use.
  • No API keys needed: Payments are handled directly with each request.
  • Network: Payments are settled over the Base mainnet.
